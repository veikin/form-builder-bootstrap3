About

Created By hongyangZhao (@zidaneaa) to help take the stress out of writing all that markup to get bootstrap forms together. it has been written using backbone.js and takes advantage of underscore.js templates. In 26 Aug 2015, it has been rewritten using bootstrap3 and chinese language.

If you have a problem, or want a specific snippet added please check out the github project. Note, this is only a simple tool so I'm not that keen on adding many features (eg. saving/exporting forms, embedding forms etc.).
